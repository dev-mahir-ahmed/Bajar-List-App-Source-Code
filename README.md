# 🛒 বাজার লিস্ট (Bajar List)

A modern, fast, and user-friendly shopping list application built with **React**, **Vite**, and **Capacitor**.

The app helps users create shopping lists, organize items by category, track completed purchases, and calculate total shopping expenses with a clean, modern and intuitive interface.

---

## ✨ Features

- 🛒 Create and manage shopping lists
- ✅ Mark purchased items as completed
- 💰 Track shopping expenses
- 📊 View total shopping cost
- 📋 Share or copy shopping reports
- 🌐 Bengali & English language support
- 💱 Multiple currency support
- 🌙 Light / Dark / System theme
- 📂 Custom categories
- 😀 Emoji support
- 📱 Fully responsive mobile UI
- ⚡ Smooth animations
- 💾 Offline support using Local Storage
- 🔒 Privacy Policy & Terms & Conditions
- 👨‍💻 Developer profile section
- 🤝 Community & Support

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/dev-mahir-ahmed/Bajar-List-App-Source-Code.git
```

Go to the project directory

```bash
cd Bajar-List-App-Source-Code
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Create production build

```bash
npm run build
```

---

## 📱 Android Build

```bash
npx cap sync
npx cap open android
```

Generate APK or AAB using Android Studio.
